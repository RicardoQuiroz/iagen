Identifica tus Archivos

Una guía rápida para saber qué es cualquier archivo y cómo abrirlo.

1

Identifica la Extensión

La extensión son las 3 o 4 letras después del punto en el nombre de un archivo. Le dice al sistema operativo qué tipo de archivo es.

Por ejemplo: documento_importante.pdf o foto_vacaciones.jpg

2

Busca en Internet

Usa tu buscador preferido para preguntar directamente por la extensión. Aquí tienes algunas ideas de búsqueda:

  • qué es un archivo .svg
  • cómo abrir un .mkv
  • programa para editar .raw
  • extensión .ai
3

Interpreta los Resultados

La búsqueda te dirá dos cosas clave:

  • El tipo de archivo: Por ejemplo, "Joint Photographic Experts Group" para un .jpg.
  • Programas compatibles: El software que puede abrir, ver o editar ese archivo.

Un Mundo de Imágenes: Visores Recomendados

Imágenes de Mapas de Bits (Raster)

Hechas de píxeles. Pierden calidad si se agrandan mucho. Son ideales para fotografías.

Extensiones comunes:

.jpg .png .gif .bmp .tiff .webp

Programas para abrirlas:

  • IrfanView (Recomendado)
  • Visor de Fotos de Windows/macOS
  • XnView
  • GIMP (Editor gratuito)
  • Adobe Photoshop (Editor profesional)

Imágenes de Vectores

Basadas en fórmulas matemáticas. Se pueden escalar a cualquier tamaño sin perder calidad. Perfectas para logos e ilustraciones.

Extensiones comunes:

.svg .ai .eps .cdr

Programas para abrirlas:

  • Inkscape (Editor gratuito)
  • Adobe Illustrator (Editor profesional)
  • CorelDRAW
  • Navegadores web (para .svg)
  • IrfanView (puede visualizar algunos formatos)

¡Atención y Cuidado!

Al descargar un programa para abrir un archivo desconocido, asegúrate de hacerlo desde su página web oficial. Desconfía de sitios de descarga no oficiales para evitar software malicioso.
